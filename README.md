# Hydrophilics2017

The System is designed to manage a Swimming Sports Tournament, Hydrophilics2017. 
This project is built using Java and MYSQL database, connecting the two using JDBC(Java Database Connectivity).

## Features:

- Handles the various events of the tournament.
- Includes the participating sportspersons' information.
- Automates the winners of the events.
- Manages the sportspersonsâ€™ travel from their accommodations to the sports venues etc.

##### Note: Achieving this objective is difficult using a manual system as the information is scattered, can be redundant and collecting relevant information may be very time consuming. All these problems are solved using this project.

## Installation-Requirements:

1. Any IDE which supports Java, preferably Eclipse
2. MYSQL
